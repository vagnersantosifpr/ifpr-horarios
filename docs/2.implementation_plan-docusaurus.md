# Melhoria de Layout CSS — Docusaurus IFPR Horários

## Objetivo

Modernizar o visual da instância Docusaurus do IFPR Horários sem quebrar a responsividade nem o menu mobile (dropdown). O design atual é funcional mas básico. Vamos elevar a experiência visual com glassmorphism, gradientes, micro-animações e tipografia premium — mantendo todas as imagens SVG já existentes na homepage.

---

## User Review Required

> [!IMPORTANT]
> As alterações são **apenas de CSS e estilo** — nenhuma lógica ou rota será modificada. A responsividade nativa do Docusaurus (incluindo menu mobile hamburger/dropdown) será **preservada e não tocada**.

> [!CAUTION]
> O `custom.css` atual contém um bloco `@media print` essencial para a **exportação/impressão da grade de horários**. Esse bloco será **obrigatoriamente preservado** na íntegra ao final do arquivo modificado. Nenhuma regra de estilo adicionada irá sobrescrever ou conflitar com ele.

> [!NOTE]
> O Docusaurus usa CSS variables próprias (`--ifm-*`). Vamos sobrescrever as que fazem sentido no `custom.css` para garantir consistência no tema claro e escuro.

## Proposed Changes

### Tema e Design System

#### [MODIFY] [custom.css](file:///c:/Users/top%20games/Documents/ParaDeploy/Horarios-IFPR-Assis/ifpr-horarios-vss/src/css/custom.css)

Melhorias:
- **Paleta verde IFPR mais rica**: tons harmonizados com HSL ao invés do verde plano
- **Navbar com glassmorphism** sutil: backdrop-filter blur + sombra suave
- **Variáveis IFM complementares**: primário, secundário, acentuação, bordas, fontes
- **Scrollbar customizada** (webkit)
- **Transições globais** suaves para links, botões e hover states

### Homepage Header

#### [MODIFY] [index.module.css](file:///c:/Users/top%20games/Documents/ParaDeploy/Horarios-IFPR-Assis/ifpr-horarios-vss/src/pages/index.module.css)

Melhorias:
- **Hero banner com gradiente animado** de fundo (verde-esmeralda → verde-musgo)
- **Partículas/formas decorativas** CSS puras (pseudo-elementos `::before`/`::after`)
- **Logotipo e título com entrada animada** usando `@keyframes` de deslize e fade-in
- **Botão CTA** com design premium: gradiente, sombra colorida, hover animado
- **Seção de anúncio** com card glassmorphism mais sofisticado
- **Responsividade**: ajustes para tablet (768px) e mobile (480px)

### Features Section

#### [MODIFY] [styles.module.css](file:///c:/Users/top%20games/Documents/ParaDeploy/Horarios-IFPR-Assis/ifpr-horarios-vss/src/components/HomepageFeatures/styles.module.css)

Melhorias:

- **Ícones SVG maiores e centralizados** com container circular sutil
